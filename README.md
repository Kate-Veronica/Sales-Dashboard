# Sales Dashboard

## Overview
This is an interactive Sales Dashboard created using Power BI. This project is all about analyzing sales performances, profits, and order trends in various regions, categories, and sub-categories. This is useful in understanding business performances as well as making decisions.

## Objective
To visualize the sales data and extract meaningful insights such as:
- Overall sales and profit performance
- Monthly sales trends
- Region-wise performance
- Category and sub-category analysis

## Tools Used
- Power BI Desktop
- Microsoft Excel/CSV (Dataset Source)

## Dashboard Features

### 🔹 KPI Cards
- Total Sales
- Total Profit
- Total Orders

### 🔹 Visualizations
- Monthly Sales Trend (Line Chart)
- Sales by Region (Bar Chart)
- Sales by Category (Pie Chart)

### 🔹 Filters (Slicers)
- Region
- Category
- Segment
- Order Date (Month)

## Key Insights
- Identified the top-performing regions based on sales
- Analyzed the monthly sales trends to understand the seasonality of sales
- Identified the most profitable product categories
- Evaluated the performance at the subcategory level
