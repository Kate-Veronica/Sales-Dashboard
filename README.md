# Sales Dashboard

## Overview
This is an interactive Sales Dashboard created using Power BI. This project is all about analyzing sales performances, profits, and order trends in various regions, categories, and sub-categories. This is useful in understanding business performances as well as making decisions.

## Objective
To visualize the sales data and extract meaningful insights such as:
- Overall sales and profit performance
- Monthly sales trends
- Region-wise performance
- Category and sub-category analysis

## Tools Used
- Power BI Desktop
- Microsoft Excel/CSV (Dataset Source)

## Dashboard Features

### 🔹 KPI Cards
- Total Sales
- Total Profit
- Total Orders
